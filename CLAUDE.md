# Claude Code Instructions - TalesMaker Pathfinding

## Overview
Этот проект реализует Baritone-inspired pathfinding систему для NPC в Minecraft моде TalesMaker.

---

## ОБЯЗАТЕЛЬНЫЕ ПРАВИЛА

### 1. Перед началом работы
```
ВСЕГДА читай TODO.md перед выполнением любой задачи!
```

### 2. Порядок работы
1. Открой и прочитай `TODO.md`
2. Найди первую невыполненную задачу `[ ]`
3. Открой `PROMPTS.md` и найди промпт для этой задачи
4. Выполни ТОЛЬКО эту задачу
5. После завершения отметь `[x]` в `TODO.md`
6. Обнови Progress Summary в `TODO.md`
7. Переходи к следующей задаче

### 3. Правила выполнения
- **Одна задача за раз** - не выполняй несколько задач сразу
- **Проверяй зависимости** - убедись что предыдущие задачи выполнены
- **Тестируй компиляцию** - после создания файла проверь `./gradlew build`
- **Не пропускай задачи** - выполняй строго по порядку
